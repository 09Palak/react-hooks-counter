# React Hooks Counter App

## Aim
To create a functional component-based React application using `useState` for logic
and `useEffect` for lifecycle events.

## Description
This application demonstrates:
- State management using `useState`
- Component lifecycle handling using `useEffect`
- Functional components with ES6 syntax

## Output
- Displays counter value
- Increments counter on button click

## Learning Outcome
- Understanding React functional components
- Using React Hooks
- Basic lifecycle management
